# NeuraCare – Smart AI Health Guard

NeuraCare is a Smart India Hackathon (SIH) healthcare innovation project focused on combining AI, IoT, and mobile technologies for real-time health monitoring and emergency assistance.

## Features
- Real-time health monitoring
- Emergency SOS alerts
- TinyML-based health prediction concept
- Firebase integration
- Mobile dashboard for vitals tracking
- AI-assisted healthcare workflow

## Tech Stack
- Kotlin
- Android Studio
- Firebase
- TinyML Concepts
- IoT Sensors
- Node.js (planned backend)

## Project Status
This repository contains the initial prototype structure and Android application developed during SIH project preparation.

## Team Project
Developed as part of Smart India Hackathon (SIH) healthcare innovation idea.
